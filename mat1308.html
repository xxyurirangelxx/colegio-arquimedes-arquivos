1. A Base de Tudo: Regra de Sinais

Para Multiplicação e Divisão:

  • Sinais IGUAIS = Resultado POSITIVO (+) | Ex: (-5) * (-3) = +15
  • Sinais DIFERENTES = Resultado NEGATIVO (-) | Ex: (-5) * 3 = -15

Para Potenciação (Base Negativa):

  • Expoente PAR = Resultado POSITIVO (+) | Ex: (-2)⁴ = +16
  • Expoente ÍMPAR = Resultado NEGATIVO (-) | Ex: (-2)³ = -8

Atenção: A Armadilha dos Parênteses!

(-3)² = +9, mas -3² = -9. Sem parênteses, o expoente afeta só o número, não o sinal.

2. Expressões Algébricas

Regras de Ouro da Potenciação:

  • Multiplicação: Mantenha a base e SOME os expoentes.
    x² * x³ = x²⁺³ = x⁵
  • Divisão: Mantenha a base e SUBTRAIA os expoentes.
    x⁵ / x² = x⁵⁻² = x³
  • Potência de Potência: Mantenha a base e MULTIPLIQUE os expoentes.
    (x²)³ = x²*³ = x⁶

3. Produtos Notáveis

  • Quadrado da Soma: (a + b)² = a² + 2ab + b²
    "O quadrado do primeiro, mais duas vezes o primeiro pelo segundo, mais o quadrado do segundo."
  • Quadrado da Diferença: (a - b)² = a² - 2ab + b²
    "O quadrado do primeiro, menos duas vezes o primeiro pelo segundo, mais o quadrado do segundo."
  • Produto da Soma pela Diferença: (a + b)(a - b) = a² - b²
    "O quadrado do primeiro menos o quadrado do segundo."

4. Equações e Inequações

Equação (=)

Busca um valor exato. Regra: trocou de lado, inverteu a operação (+ vira -, * vira /).

Inequação (<, >)

Busca um conjunto de valores. Resolva como uma equação, mas com uma regra especial:

ATENÇÃO: Ao multiplicar ou dividir os dois lados por um número NEGATIVO, você é OBRIGADO A INVERTER o sinal da inequação (< vira >).

Ex: -2x < 10 vira x > -5.

5. Propriedades dos Quadriláteros

  • Paralelogramo: Ângulos opostos iguais. Ângulos consecutivos somam 180°.
  • Retângulo: Diagonais iguais.
  • Losango: Diagonais perpendiculares (formam 90°).
  • Quadrado: O "filho perfeito". Tem tudo: diagonais iguais E perpendiculares.
